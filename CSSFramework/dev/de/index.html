Creative Framework macht das Leben für Benutzer einfacher.

Vereinfach das Programmieren

Durch vorprorammierte Klassen wollen wir das Programmieren für den Benutzer vereinfachen und den Zeitaufwand verringern. Die vorgefertigten CSS Klassen lassen sich individuel kombinieren und bieten ein Standarddesign für alle verschiedenen Componenten.

Responsive

Das Framework ist nach heutigem Responsive Standard umgesetzt und nimmt dem Benutzer die Arbeit ab sich mit Media Queries herumzuschlagen. Containerklassen als auch das Gridsystem machen es dem dem Benutzer einfach seine Webseite responsive zu gestalten

Individuelle Componenten

Wir haben beim Programmieren der Klassen besondern Wert auf das individuelle Design gelegt. So wollten wir viele unterschiedliche Gestaltungsoptionen realisieren und den Besuch jeder einzelnen Webseite zu einer einzigartigen Erfahrung werden lassen.
